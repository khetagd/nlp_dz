# RAG-ассистент по Уголовному кодексу РФ

## Описание проекта
LLM-based проект в формате Retrieval-Augmented Generation (RAG).

Цель проекта — создать интеллектуального ассистента, который сможет отвечать на вопросы по Уголовному кодексу Российской Федерации на основе текста закона.

Ассистент будет находить релевантные статьи и фрагменты УК РФ и формировать ответ, опираясь на найденный контекст. Это позволит быстро ориентироваться в нормах уголовного законодательства и получать справочную информацию по составам преступлений и ответственности.

## План реализации
В следующих этапах проекта планируется:
1. Очистка и структурирование текста закона  
2. Разбиение на смысловые фрагменты
3. Создание эмбеддингов и индекса  
4. Реализация поиска релевантных фрагментов  
5. Генерация ответа модели на основе найденного контекста
